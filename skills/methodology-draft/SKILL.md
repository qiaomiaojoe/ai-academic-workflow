# Methodology Draft · 研究设计初稿（B1-B4）

把用户**选定的方案**展开为完整研究设计与干净的 Sec 4 方法论 draft。

## 调用约定（独立运行）

- 输入：① 选定的研究设计方案（来自 `methods-map` 的 A4，或用户直接口述：路径/方法链/对标文献）② 方法地图 + methodology_plugin（项目 `02_文献/方法地图.md`；**没有时降级**：请用户给出主流方法与排除项的判断依据，或先跑 `methods-map`，所有"外挂对照"字段改标 [无外挂·用户口径]）③ RQ 与引用候选来源（`01_选题/选题.md` + `02_文献/综述初稿.md`，或用户口头给）④ 用户偏好（样本规模/时间线/软件，可选）。
- 落盘（两份文件分离 · 文件名与位置固定，不要改名/换目录/加日期后缀——下游 `article-framework` / `fulltext-draft` 按固定路径读）：项目内 `03_框架/研究设计初稿.md` + `03_框架/研究设计-设计说明.md`；项目外当前目录同名。
- 产出语言：项目 CLAUDE.md 或用户指定（英文/双语时 B4 用学术语体，正文约 1200-1500 词）。

## B1 · 全局方法决策

路径 / 结构形式 / 对标 exemplar（承接选定方案，补全理由）。

## B2 · 方法论段骨架（5-7 段），每段输出 6 维度表

`| 功能（Rationale/Context/Sampling/Collection/Variable/Analysis/Ethics/Limit） | 方法决策（选什么 · 从 mainstream_methods 里排除什么 · 为什么 trade-off） | 外挂对照（methodology_plugin 路径 · 主流/小众/新趋势/gap） | 统计决策（定量段才填：测量层级 → 合法工具链） | 引用候选（来自综述初稿 + 5 近邻，不自造） | 要写什么（3-5 句段落 outline） |`

## B3 · 研究设计清单（把决策变可执行）

1. 抽样设计：抽样矩阵（维度 × 标准 × 目标 N）+ 纳入/排除标准 + 招募路径 + 饱和判定（定性）/ 样本量合规（定量）
2. 数据收集工具：定性 → 访谈提纲（4-5 个 block，每 block 3-5 问）；定量 → 问卷结构（按 section）；录音 / 转录 / 软件细节
3. 分析框架：定性 → 编码书（演绎层框架 + 归纳层 Braun-Clarke 六步 + 整合规则）；定量 → 统计方法链（测量层级 → 工具）
4. 信度 / 效度 / 伦理：Trustworthiness 四维（Lincoln-Guba 操作化）+ Ethics 清单（IRB / 知情同意 / 匿名 / 数据存储 / 退出权利 / 风险披露）+ Reflexivity（作者立场 + bracketing）
5. 时间线：从 IRB 到 draft 的阶段 × 时间估计（结合用户预期时间线）

## B4 · Sec 4 方法论 draft（合成）

基于 B2 + B3 合成 Research Methodology draft：所有具体数字（N / κ / 时间线）来自 B3；关键节点写成"选 A 排除 B"的方法决策叙事；引用 author-date。⚠️ B4 正文是落盘到「干净初稿」的内容，正文里**不要夹** [骨架对照] / [外挂引用] / [教学点] 等批注——这些放辅助文档。

**一致性检查（最后必做）**：Sec 4 是否呈现 operationalisation（虚→实）；每段是否有外挂对照；每段方法决策是否可追溯到选定方案。

## 硬约束与落盘

- 不编造文献/数字：[待补] / [TBD-citation: 角色] 占位；排除的备选方法必须出自 mainstream_methods；不用综述/外挂完全没涉及的方法；limitation 不写成诉苦。
- `研究设计初稿.md` = **干净版 · 下游输入**：只放 B4 正文（连续可读成稿，零批注）。
- `研究设计-设计说明.md` = **辅助文档**：B1 + B2 骨架表 + B3 清单 + 每段 [教学点]（方法论写作技法 1-2 句）+ [骨架对照]/[外挂引用] 对照 + 一致性检查。
- 两份文件顶部互相指引一行。交付前自检：文件名位置精确一致 / 初稿真干净 / 说明与初稿逐段对应 / 占位符齐全。
