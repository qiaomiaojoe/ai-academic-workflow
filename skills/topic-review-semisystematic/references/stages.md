# 综述② 半系统 / 叙事综述 · 三阶段详细步骤

方法锚：Snyder (2019), *Journal of Business Research*, 104, 333-339.
范例锚：McColl-Kennedy et al. (2017). The changing role of the health care customer. *Journal of Service Management*, 28.

**范例预览（一次性端给用户看，作全貌参照）**：McColl-Kennedy et al. (2017) 面对"医疗顾客角色"这个跨学科大主题 → 梳理不同研究传统怎么理解它 → 综合成主题版图 → 给出一份**研究议程**。这就是半系统综述的样子：主题宽、跨传统、透明但不穷尽、产出脉络与议程。

## STAGE A · 挖掘（理清 研究对象 / 范围 / 问题）

> 三型通用。教练**先给 2-3 个草案**让用户选 / 改，一步一问。**不要求用户诊断"文献是不是很乱"**——文献的张力 / 缺口由教练在 A.4 帮点出。本型 A.2/A.3 侧重接受"宽而有边界"的问题，不强求收窄。

- **A.1 研究对象**：据用户输入给 2-3 个"到底要梳理什么"的更聚焦界定（一片领域 / 一个跨学科主题），用户选 / 改。
- **A.2 情景 / 范围**：给 2-3 个范围候选——涉及哪些学科 / 研究传统、大致时间窗、纳入到什么程度。
- **A.3 研究问题（雏形）**：给 2-3 个"领域地图式"的宽问题草案（有边界，但不强求收窄）。顺带判断：若问题其实很窄、要证据定论 → 提示可能更适合 `topic-review-systematic`；若要造新框架 → `topic-review-integrative`。
- **A.4 为什么值得做 + so-what**：帮用户想清"为什么现在需要这篇综述、谁会因此不同"。**教练主动点出这片文献可能的张力**（如碎片化、各学科各做各的、概念混乱、老框架跟不上新现象）作为佐证——教练的判断输入，不是用户门槛。产出 ① 理论增量句 ② 现实利害句（落到具体人群 / 机构）。
- **A.5 挑战**：这问题该用综述回答，还是其实该做一篇实证？已有综述是否已覆盖？你的差异在哪？
- **A.6 输出** → 汇总写入 `选题-A-mining.md`（中间产物 · 断点续跑用；中英对照则同文件双语并排）。

## STAGE B · 校准（半系统特化）

- **B.1 组织性问题**：给 2-3 个"领域地图式"的宽问题（有边界，但不强求收窄），用户选 / 改。
- **B.2 组织框架 / 组织原则**：给 2-3 种组织方案——按主题 / 按历时发展 / 按研究传统 / 按论争——各附 trade-off，用户选。这决定综述怎么搭骨架。
- **B.3 边界与检索策略（透明但非穷尽）**：定纳入哪些研究传统、时间窗、代表性取样逻辑——**要透明到读者能判断合理性**（Snyder：半系统不追求穷尽，但过程要可评估）。产出检索种子给下一步文献搜索。
- **B.4 综合方法**：主题分析 / 内容分析 / meta-narrative 里选一个（下游数据分析走定性）。
- **B.5 近邻综述对位**：让用户列 2-3 篇已有评述 / 领域地图（真实、可核对），说你的差异化角度 + 你要给的研究议程雏形。
- **输出** → 写入 `选题-B-calibration.md`。

## STAGE C · 输出

### C.0 综述选题压力测试（Snyder 质量四维 depth / rigor / replicable / useful + §5 坑；半系统重点）
- 通用四问：① 贡献是不是真"新"（不是描述性汇总 / 词云 / 引文统计）？② 范围有没有砍太狠？③ 类型-问题-分析法三者匹配吗？④ 透明度够不够？
- 半系统特化：取样虽非穷尽，是否透明到读者能判断合理性？meta-narrative 会不会流于罗列？给的研究议程是不是真有洞见（不是"还需更多研究"这种空话）？
- 不过关 → 打回 Stage B 对应步骤。

### C.1 产出 `选题.md`
顶部机器锚块（见 SKILL.md 产出规则）+ 正文：
- **【综述规格块】**：综述类型 semi-systematic / 组织性问题 / 组织框架 + 组织原则 / 边界与检索策略（透明但非穷尽）/ 综合方法（主题/内容分析）/ 预期贡献（现状 / 主题 / 历史脉络 / 研究议程 + "新在哪"）/ 报告规范 RAMESES / 分析走向（定性 → 数据分析步骤）。
- **【PXYV 派生投影】**：P=组织性视角/元叙事；X=被梳理的主题/研究体；Y=跨学科领域+时间窗；V=脉络图/主题/研究议程。（只为下游 `pxyv-search` 兼容，由综述规格映射得出。）
- **近邻综述表**：B.5 的 2-3 篇，真实可核对。
