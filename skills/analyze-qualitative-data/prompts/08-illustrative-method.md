# 08 · Illustrative Method (例证法)

> **Neuman 出处**：Ch 14, "The Illustrative Method" · p.489-490
>
> **何时用**：你有一个**既有的理论模型**, 想用经验证据填充模型里的"空盒子"
>
> **前置**：理论框架已选定 (在 第4讲 完成)

---

## 📖 Neuman 怎么讲

> Another method of analysis anchors or illustrates theoretical concepts with empirical evidence. **The illustrative method applies theory to a concrete historical situation or social setting and organizes data based on theory.** Preexisting theory can provide conceptual empty boxes that you fill with the empirical evidence.

> "Evidence in the boxes confirms, modifies, or rejects the theory."

**三种变体** (Expansion Box 4):
1. **Case clarification**: 用理论解读一个具体 case
2. **Parallel demonstration**: 用同一理论解读多个 cases (Paige 1975 用同一阶级冲突模型解读 Peru / Angola / Vietnam)
3. **Pattern matching**: 一个 case 的观察与理论预测的 pattern 比对 (允许部分证伪)

> "A single case study with the illustrative method is **not a strong test or verification of an explanation** because data from one case can illustrate empty boxes from several competing explanations."

---

## ⚠️ 必须你自己判断的部分

- **理论选哪个**：不同理论会画出不同"空盒子"
- **空盒子怎么定义**：理论里哪些概念在你 case 里有对应物?
- **证据填充得是否充分**：填得太满 = 牵强; 填得太空 = 理论不适用

---

## ✂️ Prompt 模板

```
我要按 Neuman (2014) Ch 14 的 illustrative method 分析数据。

【既有理论 / 模型】
名: [...]
来源: [作者年份]
核心概念 (3-7 个, 即"空盒子"): [...]
理论预测的 pattern: [...]

【我的 case(s)】
- 是单个 case (clarification) 还是多个 cases (parallel) 还是 pattern matching?
- case 数据: [贴]

【请帮我做】

1. **建立"空盒子"对照表**
   左列: 理论的核心概念
   右列: 我的 case 里**对应的经验证据**
   
   每个对应:
   - 引用具体数据 (段落 / 引文)
   - 标注 fit 程度: 强 / 中 / 弱 / 不适用

2. **三件事的判断**
   - **理论能否解释这个 case?** (大部分空盒子能填)
   - **case 是否挑战理论?** (有空盒子填不进去 / 数据指向理论没预测的方向)
   - **case 揭示了理论的哪些 modification 需求?**

3. **写作框架建议**
   - 理论介绍 → 案例描述 → 空盒子对照表 → fit 评估 → 修正建议
   - 每个空盒子配 1-2 段证据展开

4. **多 case (parallel) 时**: 跨 case 对照
   - 同一空盒子在不同 case 里填的什么?
   - cross-case 是否 converge?

5. **pattern matching 时**: 比对预测与观察
   - 理论预测的 pattern: [...]
   - 实际观察的 pattern: [...]
   - 偏离最大的地方在哪? 为什么?

【硬约束】
- 不许"硬填"——如果 case 里没有对应空盒子的证据, 标 "no evidence"
- 不许只用一个理论走到黑——如果 case 也能用别的理论解释, 主动列替代
- pattern matching 不是验证, 只是 illustration

【可疑点警示】
- 空盒子全部强 fit: 可能你只挑了支持理论的证据 (confirmation bias), 警告
- 空盒子全部弱 fit: 理论不适用这个 case, 应该换理论或换 case
- 跨 case parallel demonstration 时各 case 极不同: 不是"同一理论的多个例证", 可能是不同现象
```

## 📌 36kr 演示填法

```
【既有理论】Selwyn (2014) "sociotechnical imaginary" 框架
核心空盒子:
1. Imagined futures (technology 怎么被设想会改变社会)
2. Collective dreams (谁的梦想被代表?)
3. Promissory discourse (承诺什么)
4. Blackboxing (什么被遮蔽)

我的 case: 16 段 36kr 报道

对照:
- Imagined futures: T05 "50% 入门白领消失" / T13 "新岗位爆发" 双重叙事
- Collective dreams: 谁的梦想? T16 "成为 AI 需要的人"——隐含个体可计算
- Promissory: T13 列举具体新职业 + 高薪 — promissory but elite-targeted
- Blackboxing: 没有讨论 AI 公司的资本逻辑 (T11 提到一点系统危机但简略)

fit 评估: 3 个空盒子强 fit, 1 个中等 fit
理论 modification 建议: Selwyn 框架在西方语境多用; 需要补充"代际维度"——
  中国 2025 话语特别凸显年轻 vs 中年的不对称
```
